# Сервис для аренды вещей 

Данный проект представляет собой сервис, где можно ареновать вещи на определенный срок или выставить их на аренду.Люди сталкиваются с разовой задачей, нуждаются в доступном способе получить нужную вещь без покупки, так как её использование длится всего несколько часов или дней, а покупка обходится дороже самой задачи. Именно эту проблему будет решать наш проект.

- На Android/iOS :
  - Запустите приложение из главного экрана
