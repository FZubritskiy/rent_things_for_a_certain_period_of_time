# Сервис для аренды вещей 

Данный проект представляет собой сервис, где можно ареновать вещи на определенный срок или выставить их на аренду

## Разработчики : FZubritskiy , zacguleykov-cyber, iliamuravyev , a-da-nu-nafig.

## Технологический стек :
- HTML
- CSS
- JavaScript
- Python
- MySQL

# Как запустить :
- На Windows/Linux/MacOS:
  - Запустите исполняемый файл rent_things_for_a_certain_period_of_time для своей операционной системы или  или перейдите по ссылке : rent_things_for_a_certain_period_of_time.com
- На Android/iOS :
  - Запустите приложение из главного экрана
